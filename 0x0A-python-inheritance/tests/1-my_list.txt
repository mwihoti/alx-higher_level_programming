How to use 1-my_list.txt
=========================

This function prints list in sorted order

Numbers
	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()
	>>> isinstance(my_list, list)
	True
	>>> my_list.print_sorted()
	[]

	:: list with numbers
	>>> my_list.append(3)
	>>> my_list.append(6)
	>>> my_list.append(1)
	>>> my_list.print_sorted()
	[1, 3, 6]

	:: with zero value
	>>> my_list.append(0)
	>>> my_list.print_sorted()
	[0, 1, 3, 6]

	:: with negative values
	>>> my_list.append(-1)
	>>> my_list.print_sorted()
	[-1, 0, 1, 3, 6]

	:: with same numbers
	>>> my_list.append(1)
	>>> my_list.print_sorted()
	[-1, 0, 1, 1, 3, 6]
